# 🎨 Color Palette Generator

## Overview
Color Palette Generator is a simple web application that helps users generate beautiful color palettes for web design, graphic design, and creative projects. Users can generate random colors and copy color codes for use in their applications.

## ✨ Features
- Generate random color palettes
- Display multiple colors at once
- View HEX color codes
- Copy color codes with a single click
- Responsive and user-friendly interface
- Quick inspiration for designers and developers

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🚀 How It Works
1. Open the application.
2. Click the **Generate Palette** button.
3. A new set of colors will be displayed.
4. Copy any color code and use it in your projects
